## 📋 Program Overview

This intensive program builds a complete **.NET Full Stack Developer** skillset, covering:

### Phase-wise Coverage
| Phase | Modules |
|-------|---------|
| **1. Core .NET & C#** | Visual Studio, C# Fundamentals, OOP, Collections, Generics, Advanced C# (7.0/8.0), Exception Handling, File I/O, Delegates, Reflections, Unit Testing |
| **2. SQL Server** | Database Design, DDL/DML/DCL/TCL, Joins, Subqueries, Triggers, Indexes, Views, Normalization |
| **3. ASP.NET Core (Razor + MVC)** | Razor Pages, MVC Architecture, Model Binding, Tag Helpers, Validations, Cookies & Sessions, Filters, ADO.NET, EF Core (Code-First & DB-First), Repository Pattern, CRUD Operations |
| **4. Security** | Input Validation, Output Encoding, Authentication & Authorization, JWT + OAuth, Database Security, Secure SDLC Practices |
| **5. Web API & Microservices** | REST Architecture, Postman, Microservices vs Monolithic, Ocelot API Gateway |
| **6. Cloud & Azure** | Docker, AKS (Kubernetes), Azure Functions (Serverless), APIM, Event Grid, Redis Cache, Azure App Services, App Configuration, Key Vault |
| **7. DevOps** | Git/GitHub, CI/CD (GitHub Actions + Azure DevOps), Merge Control |
| **8. Angular** | Angular CLI, Modules, Components, Templates, Data Binding, Services, Routing, Forms, HTTP Client |
| **9. Capstone Project** | Agile Sprint Planning, Development & Review (Full-stack application) |
| **10. Modern Tools** | GitHub Copilot, Agentic AI, Prompt Engineering, Responsible AI, Microsoft 365 Copilot, RAG Basics |
